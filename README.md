# vessel-analyzer
Browser-based vessel measurement and angiography analysis tool

Vessel Analyzer is a browser-based research tool for quantitative analysis of coronary microvasculature from angiography image sequences. It uses a Frangi vesselness filter (Frangi et al. 1998, Medical Image Computing) and runs entirely in the browser using Pyodide and scikit-image. The software was developed by Vasco Sequeira, PhD, for research use only and is not intended for clinical decision-making.
